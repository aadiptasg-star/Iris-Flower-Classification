# Iris Flower Classification

## Project Overview

This machine learning project classifies Iris flowers into three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

using flower measurements such as sepal length, sepal width, petal length, and petal width.

## Dataset

The Iris dataset from the UCI Machine Learning Repository.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow

1. Data Loading
2. Data Exploration
3. Data Visualization
4. Train-Test Split
5. Logistic Regression Model
6. Model Evaluation
7. Confusion Matrix
8. Classification Report

## Results

Accuracy Achieved: 100%
